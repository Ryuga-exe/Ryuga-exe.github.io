Starting from APR 6, 2022, the PC launcher will automatically install relevant components if you are using Windows 7. ★ PC Launcher Extension Installation FAQs Q1: PC launcher indicates "The extensions required for running the game failed to be installed (1058)".

A1:
1. Hold down Windows Key and press R to open "Run" window, where you enter services.msc and click OK to open "Service" window.
2. In the "Service" window, double-click on "Windows Update".
3. In the pop-up window, select Manual for "Startup type", then click Apply and OK.
4. Exit Honkai Impact 3rd PC launcher (Make sure you exit the program, not minimize to system tray).
5. Restart the Honkai Impact 3rd PC launcher and it will automatically install the patches required.
6. After successfully installing, reboot the computer as directed by the pop-up window.

Q2: PC launcher indicates that "The extensions required for running the game failed to be installed (-2145124297)".

A2:
1. Go to https://www.microsoft.com/en-us/download/details.aspx?id=46148, and download the patch Windows6.1-KB3033929-x64.
2. Change the suffix of Windows6.1-KB3033929-x64.msu to Windows6.1-KB3033929-x64.cab, and decompress it into a new folder (do not decompress into the default download folder).
3. Hold down Windows Key and press R to open "Run" window, where you enter cmd and click OK to open Command Prompt.
4. In Command Prompt, enter dism /online /add-package /packagepath: "X:\route\filename.cab" and then press Enter to install.
(For example, if you decompressed files to the "update" folder of Disk D, then please enter dism /online /add-package /packagepath:"D:\update\Windows6.1-KB3033929-x64.cab")

Q3: PC launcher does not respond when clicked.
A3:
1. Double-click on launcher.exe in the file directory and observe if there are any dump files generated in the launcher's file directory. If so, please include the dump file and the logs folder of launcher installation file directory in a new folder. Send the new folder to our customer service as feedback (only send logs folder if there are no dump files).
2.In the launcher installation file directory, double-click on "vcredist2013_x64.exe" to install. Go to Microsoft's official website below and download and install the latest VC_redist.x64.exe of the corresponding Visual Studio version. Reboot the gamehttps://docs.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170
Q4: After opening the PC launcher, its main screen glitched (for example, the display is cut off). A4: Right-click on the PC launcher icon of Honkai Impact 3rd to open Properties. Select [Compatibility] - [Choose the high DPI settings for this program], and then select [High DPI scaling override] and [Application]. Click Apply and OK. Reboot the PC launcher and the glitch will be fixed.
